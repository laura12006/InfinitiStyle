HU01 - Registro
================

**Como persona interesada en la aplicación**, quiero poder registrarme fácilmente, para comenzar a explorar y buscar prendas en ella.

Pasos del usuario
-----------------
1. Abrir la aplicación Style Infinite.
2. Seleccionar la opción **“Registrarme”** en la pantalla principal.
3. Llenar el formulario con:
   - Nombre completo
   - Correo electrónico
   - Contraseña
4. Aceptar los términos y condiciones.
5. Pulsar **“Crear cuenta”**.

Criterios de aceptación
-----------------------
- Todos los campos son obligatorios.
- Si el correo ya existe, debe mostrar un mensaje de error.
- Al registrarse correctamente, el sistema redirige al catálogo de prendas.
